# Appendix A: Messaging Framework

## Elevator Pitch (30 seconds)

> ADP is an open-source debug panel for PHP that works with Laravel, Symfony, Yii, and any other framework. Unlike Telescope or Symfony Profiler, which are locked into a single framework, ADP provides a unified interface for all your projects. Plus unique features: live application introspection, microservices support, and even debugging Python/Node services via a language-agnostic API.
