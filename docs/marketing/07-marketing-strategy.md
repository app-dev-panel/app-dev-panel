# 7. Marketing Strategy

## 7.1 Positioning

**Tagline**: *"The Universal PHP Debug Panel"*
**Sub-tagline**: *"One panel for Symfony, Laravel, Yii — and beyond"*

**Positioning statement:**
> ADP is the only open-source platform for debugging and inspecting PHP applications that works with any framework. Unlike Telescope (Laravel only) or Symfony Profiler (Symfony only), ADP offers a unified, modern UI with 28 collectors, live application introspection, and microservices support.

## 7.4 Launch Campaigns

### Campaign 1: "The Great Migration" (Laravel focus)

**Goal**: attract Laravel developers (the largest PHP community)
**Format**: series of 3 articles + video
1. "What ADP does that Telescope can't" — feature comparison
2. "Migrate from Telescope to ADP in 10 minutes" — step-by-step guide
3. "ADP Inspector: see your Laravel app like never before" — Inspector demo

### Campaign 2: "Debug Everything" (polyglot focus)

**Goal**: attract polyglot developers and microservices teams
**Format**: demo + blog post
1. PHP backend + Python ML service + Node.js API — all in one ADP
2. Architecture diagram + setup guide

### Campaign 3: "Framework Freedom" (agency focus)

**Goal**: agencies with projects on different frameworks
**Format**: case study + LinkedIn posts
1. "How [Agency] standardized debugging across 20 projects"
2. ROI calculator: time saved on onboarding
