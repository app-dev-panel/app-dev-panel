# 1. Executive Summary

ADP (Application Development Panel) is the only **framework-agnostic** PHP debug panel on the market, supporting Yii 3, Symfony, Laravel, and Yii 2 out of the box. Unlike competitors (Laravel Telescope, Symfony Profiler, Clockwork, Ray) locked into a single framework, ADP provides a unified interface for any PHP project.

## Key Differentiators

| Factor | ADP | Competitors |
|--------|-----|------------|
| Framework support | 4 frameworks + language-agnostic API | 1 framework each |
| Real-time streaming | UDP + SSE | HTTP polling only |
| Inspector (live state) | 20+ introspection pages | No equivalent |
| Multi-app debugging | Service Registry | None |
| Cost | Open Source (free) | Ray — paid, Blackfire — paid |
| Language-agnostic ingestion | OpenAPI 3.1 + Python/TS clients | PHP only |

## Core Thesis

> **"One panel for all your PHP apps. And beyond."**

ADP is not another debugbar. It is a full-fledged **developer platform** combining debugging, introspection, monitoring, and tooling (Git, Composer, Code Generation) in a single UI.
