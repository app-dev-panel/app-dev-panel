# 4. Feature Analysis & USPs

## 4.1 Killer Features (Top 5 Selling Points)

**KF-1: "One Panel, Any Framework"**
- Message: *"Switch projects, not tools"*
- A developer working across multiple frameworks gets one tool
- Promotion: screencast switching between Laravel and Symfony in a single ADP

**KF-2: Inspector — Live App Introspection**
- Message: *"X-ray your running app"*
- 20+ live introspection pages: routes, DB schema, DI, config, Git, files
- No competitor offers this
- Promotion: demo video "everything about your app without the CLI"

**KF-3: Language-Agnostic Ingestion API**
- Message: *"Debug anything. PHP, Python, Node — one panel."*
- OpenAPI 3.1 specification + Python/TypeScript clients
- Promotion: example with PHP + Python ML service in a single ADP

**KF-4: Multi-App Service Registry**
- Message: *"Microservices? One debug panel for all."*
- Services register themselves, inspection is routed automatically
- Promotion: microservices architecture diagram

**KF-5: Developer Toolkit (Git, Composer, GenCode, cURL)**
- Message: *"Your IDE's best friend"*
- Code generation, package management, Git, cURL, request replay
- Promotion: workflow video
