Variant 14 — Timeline Waterfall

Request lifecycle as a horizontal waterfall chart

EXPERIMENTAL DEVTOOLS-STYLE PERFORMANCE

The toolbar itself IS the visualization. Collapsed: a proportional color bar showing how the request time was spent. Expanded: a full waterfall chart like Chrome DevTools Network tab, but for server-side phases.

  • Collapsed bar shows proportional time per phase (boot, routing, controller, DB, HTTP, view, response)
  • Expanded waterfall with labeled rows and time ruler
  • Each DB query and HTTP call shown as individual bars
  • Click a row for detail popover
  • Instant visual answer to "where did the time go?"

Interaction: Click the timeline bar → expands to waterfall → click rows for detail → click ▾ to collapse.
