Variant 15 — Ambient Glow Border

Page edge color reflects application health

EXPERIMENTAL AMBIENT ZERO UI

The most minimal possible toolbar: a thin 3px colored border around the entire viewport. Green = healthy, yellow = slow, red = error, blue = loading. No buttons, no text — just color. A tiny corner badge shows details on hover.

  • 3px glow strip on all 4 edges — adapts color to request health
  • Green glow = fast, healthy response
  • Yellow glow = slow response (>200ms) or warnings
  • Red glow = exception or 5xx (pulses!)
  • Blue sliding animation = loading/in-progress
  • Tiny corner badge with duck + status code + time
  • Click badge for detail popup
  • Absolute minimum visual intrusion
