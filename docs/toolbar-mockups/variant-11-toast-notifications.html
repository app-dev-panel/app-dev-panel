Variant 11 — Smart Toast Notifications

Event-driven debug notifications

RADICAL NON-INTRUSIVE EVENT-DRIVEN

Instead of a persistent toolbar, debug insights appear as contextual toast notifications. Only shows what matters — slow queries, errors, deprecations, N+1 problems. Quiet when everything is fine.

  • Request summary toast on every page load (auto-dismisses after 5s)
  • Warning toasts for slow queries, N+1 detection, deprecations
  • Error toasts for exceptions and failed requests
  • Click to expand and see details inline
  • Auto-dismiss with progress bar (hover pauses countdown)
  • Tiny pill at bottom-right shows persistent status
  • Zero visual noise when everything is healthy
