  • Code blocks, metric badges, action buttons inline
  • Typing indicator for async analysis
  • Could connect to real MCP server for actual AI analysis

Interaction: Click the duck FAB → chat opens with auto-summary → type questions or click suggestion chips → duck responds with debug insights.
