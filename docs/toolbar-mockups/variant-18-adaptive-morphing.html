HYBRID V1+V3 MORPHING

The same toolbar morphs between bottom bar (V1 style) and side rail (V3 style) with a single smooth CSS transition. Layout, metrics, and interactions adapt automatically.

  • Bottom mode: horizontal chips, slide-up animation from V1
  • Side mode: vertical rows with labels, like V3
  • 0.45s spring transition between modes — everything morphs
  • Collapsed pill repositions to match current mode
  • Side mode has resize handle on the edge
  • Panel area slides up (bottom) or doesn't appear (side — uses full height)
  • Toggle with buttons or could be user preference

Interaction: Click pill to expand → use position buttons (top-right) to switch modes → smooth morph animation → click 🦆 to collapse.
