Variant 3 — Side Rail

Vertical toolbar panel on the right edge

IDE-STYLE VERTICAL SPACE-EFFICIENT

A vertical side panel that slides out from the right edge. Inspired by IDE tool windows (JetBrains, VS Code side panels). Metrics are displayed as rows with icon + label + value.

  • Collapsed: a small tab on the right edge (like IDE tool window buttons)
  • Expanded: a 280px side panel with structured metric rows
  • Each metric row has hover highlight with colored left border accent
  • Request info displayed prominently at top
  • Vertical layout allows more info per metric (label + value side by side)
  • Doesn't interfere with bottom content (forms, footers, sticky elements)
  • Resizable via left edge drag handle

Interaction: Click the tab on right edge → side rail opens → hover metrics → click actions at bottom → click ✕ to close.
