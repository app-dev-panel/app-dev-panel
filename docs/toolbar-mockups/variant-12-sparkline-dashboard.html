Variant 12 — Sparkline Dashboard

Mini charts showing metric trends across requests

EXPERIMENTAL DATA-RICH LIVE

Each metric card contains a sparkline chart showing its value over the last 10 requests. At a glance you see not just the current value but the trend — is response time going up? Are queries increasing?

  • SVG sparkline charts rendered inline on each metric card
  • Trend indicator: ↑ worse / ↓ better vs previous request
  • Glow hover effects per metric color
  • Live pulse dot — auto-updates when new requests arrive
  • "Simulate request" button generates new random data + flash animation
  • Dark-native dashboard aesthetic

Interaction: Click trigger → dashboard opens → hover cards for glow → click "Simulate Request" to add data points with flash animation.
