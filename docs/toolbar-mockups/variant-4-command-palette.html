Variant 4 — Command Palette

Spotlight / ⌘K style debug toolbar

RADICAL KEYBOARD-FIRST NON-INTRUSIVE

Completely different approach: the toolbar is not always visible. Instead, there's a tiny status bar at the bottom (like VS Code) and a keyboard shortcut triggers a command palette overlay.

  • VS Code-like mini status bar at the very bottom (24px) — shows key metrics inline
  • ⌘D / Ctrl+D opens a full command palette overlay
  • Searchable — type to filter metrics and actions
  • Keyboard navigation (↑↓ arrows, Enter to select)
  • Request summary prominently displayed at top of palette
  • Zero visual overhead when not actively debugging
  • Combines toolbar + entry selector + panel launcher in one UI

Interaction: Press Ctrl+D or click the 🦆 in the status bar → palette opens → type to search → navigate with arrows → Esc to close.
