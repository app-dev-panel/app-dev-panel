RADICAL RETRO HACKER

A draggable terminal window with CRT scanline effect, green phosphor text, and an interactive command prompt. Debug data presented as terminal output — type commands to drill into specific collectors.

  • CRT scanline overlay + blink cursor for authentic retro feel
  • Interactive prompt — type "db", "logs", "events" to see details
  • Auto-scrolling output log with color-coded severity
  • ASCII box drawing for structured data
  • Matrix-green color scheme, phosphor glow effects
  • Nostalgic + functional — devs love terminal aesthetics

Interaction: Click the trigger (bottom-right) → terminal opens → type commands (help, db, logs, clear) → Enter to execute → click red dot or Esc to close.
