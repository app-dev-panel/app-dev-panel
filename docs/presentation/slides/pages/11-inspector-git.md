# Git прямо из инспектора

Когда-то я работал в компании, где сервисы деплоились на отдельные стейдж-сервера с контейнерами.

Чтобы поменять ветку — нужно было **ждать деплой**.

<v-click>

Поэтому я сделал **git прямо в инспекторе**:

- branch / log / checkout
- pull / fetch
- без ssh, без деплоя, без слёз

</v-click>

<v-click>

А для `runtime/var` добавил файловый браузер — чтобы не лезть по ftp.

</v-click>
