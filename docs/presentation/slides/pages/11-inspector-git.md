# Git прямо из инспектора

- branch / log / checkout
- pull / fetch
- файловый браузер по `runtime/`

<!--
Speaker notes:
Когда-то я работал в компании, где сервисы деплоились
на отдельные стейдж-сервера в контейнерах. Чтобы
поменять ветку — нужно было ждать полный цикл деплоя.
Это 5-15 минут на каждый чекаут. Очень больно.

Поэтому я сделал git прямо в инспекторе: branch, log,
checkout, pull, fetch — всё через UI, без ssh, без
деплоя, без слёз.

А для папки runtime/ добавил файловый браузер — чтобы
не лезть по ftp каждый раз, когда нужно посмотреть
лог или cache-файл.

Это главный принцип инспектора: кнопки берутся из
реальных болей, а не из воображаемых «а вот было бы
хорошо, если бы».
