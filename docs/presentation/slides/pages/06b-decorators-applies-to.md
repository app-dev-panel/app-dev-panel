# Один подход — любой сервис

| PSR | Что оборачиваем |
|---|---|
| PSR-3 | логгер |
| PSR-11 | DI-контейнер |
| PSR-14 | event dispatcher |
| PSR-18 | HTTP-клиент |
| PSR-16 | кэш |

Пользовательский код об этом не знает.
