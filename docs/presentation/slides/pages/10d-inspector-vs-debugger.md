# Дебаггер vs Инспектор

<div class="grid grid-cols-2 gap-10 mt-8">

<div>

### Дебаггер — прошлое

- собирает в момент запроса
- кладёт в хранилище
- показывает потом

> «что **было**?»

</div>

<div>

### Инспектор — настоящее

- ничего не копит
- спрашивает в момент открытия UI
- показывает прямо сейчас

> «что **есть**?»

</div>

</div>
