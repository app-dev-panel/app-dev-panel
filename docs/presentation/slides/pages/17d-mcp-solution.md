# Шаг конём: MCP + свой ИИ-клиент

- MCP-сервер — агент берёт данные напрямую, без логов
- ИИ-клиент прямо в панели — не прыгаешь между вкладками
- OpenRouter — если нет платного API
- Claude с подпиской — тоже работает
