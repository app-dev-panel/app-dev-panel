# ИИ пришёл — дебажить стало больнее

- агент может читать логи — но это медленно
- логи — один огромный файл, грепать больно
- может схватить старые данные вместо свежих
- данные реквеста исчезают, когда реквест заканчивается
- руками: обёртки, брейкпоинты, ssh в контейнер

дебаг-панели работают как часы — но морально устарели
