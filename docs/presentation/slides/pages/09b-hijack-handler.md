# Тогда не надо нового API

Перехватываем **существующий** handler — как это делает Buggregator.

```text
dump($user)
    ↓
наш var-dump handler
    ├─→ наш коллектор          ───→  ADP панель
    └─→ оригинальный handler   ───→  Buggregator / Xdebug / ничего
```

<v-click>

- Пользователь пишет `dump($x)` как обычно
- Ничего не импортирует, ничего не устанавливает
- Данные прилетают **и** к нам, **и** туда, где он привык их видеть

</v-click>
