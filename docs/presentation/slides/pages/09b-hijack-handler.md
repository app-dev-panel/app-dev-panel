# Перехватываем существующий dump handler

```text
dump($user)
    ↓
наш var-dump handler
    ├─→ коллектор ADP
    └─→ оригинальный handler → Buggregator / Xdebug
```

- ноль нового API
- ноль импортов
- данные идут и к нам, и туда, где их ждут
