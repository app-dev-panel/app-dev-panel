# Доменные события — ещё интереснее

Если мы знаем **конкретное** событие — можно выжать из него пользу.

| Событие | Что получаем |
|---|---|
| `UserLoggedIn` | айди юзера, роли, guard |
| `RouteMatched` | какой контроллер, какой action |
| `CacheMiss` | ключ, driver, время |
| `QueryExecuted` | SQL, байндинги, duration |

Не «что-то произошло», а **кто, где, зачем**.
