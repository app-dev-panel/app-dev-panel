# 06 — Compare Mode (Horizontal Split)

## Entering Compare Mode

Compare mode activates when:
1. Right-click an entry -> "Compare with..." -> select second entry
2. Ctrl+Click a second entry while one is already selected
3. Keyboard: select entry, press `C`, then select second entry

## Swap Entries

Button to swap A and B positions:

```
┌─ A: POST /api/users  201  45ms ───────────────────────────── [Swap A/B] ──┐
│  ...                                                                       │
├────────────────────────────────────────────────────────────────────────────── │
├─ B: PUT /api/users/5  200  38ms ──────────────────────────────────────────┤
│  ...                                                                       │
└────────────────────────────────────────────────────────────────────────────┘
```

## Exiting Compare Mode

```
[Exit Compare]  button in the top-right of content area
OR
Escape key
OR
Click any single entry in the list (exits compare, selects that entry)
```
