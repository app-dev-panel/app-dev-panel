# Variant D: Minimal Zen — Debug Main Dashboard

## Concept

The debug dashboard is a cards grid. Each collector is a card showing its icon, name, count badge,
a one-line summary, and a tiny sparkline. Three cards per row on desktop. Click a card to expand it
inline — the expanded card pushes siblings down and reveals the collector detail panel. Click again
(or press Escape) to collapse.

│  ⬡  Request                  1  │     │  Icon: 20x20, left-aligned
