# Variant D: Minimal Zen — Inspector: Configuration

## Concept

Configuration is displayed as a searchable card list. Each configuration group is a card.
Cards are full-width and stacked vertically. Each card shows the group name, parameter count,
and a preview of key values. Click to expand and see all parameters. Search filters cards in
real time.

## Search — Filtering Cards and Parameters

Typing in the search box filters both card names and individual parameter keys/values.
Non-matching cards collapse. Matching parameters are highlighted.

## Value Type Rendering

```
String:     "My App"                 → quoted, default color
Number:     5432                     → unquoted, accent color
Boolean:    true / false             → unquoted, green/red
Array:      ["localhost", "127..."]  → brackets, truncated
Object:     {driver: "pgsql", ...}   → braces, truncated
Hidden:     ***hidden***             → muted, italic
Null:       null                     → muted, italic
```
