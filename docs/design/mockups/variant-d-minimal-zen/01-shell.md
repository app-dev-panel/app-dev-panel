# Variant D: Minimal Zen — Shell

## Concept

The shell is almost invisible. A single thin top bar holds the logo, the entry selector pill,
and a handful of icon buttons. Everything below is content — full width, no sidebar, no chrome.
The command palette (Ctrl+K) replaces all traditional navigation.

## Transition Animations

```
Page Change (slide-left, 200ms ease-out):
  ┌──────────┐            ┌──────────┐
  │ Page A   │ ──slide──▶ │ Page B   │
  │          │            │          │
  └──────────┘            └──────────┘

Command Palette (fade + scale, 150ms ease-out):
                     ┌──────────┐
  (backdrop fade) →  │ Palette  │  ← (scale from 0.95 to 1.0)
                     └──────────┘

Entry Switch (crossfade, 150ms):
  ┌──────────┐            ┌──────────┐
  │ Entry A  │ ──fade───▶ │ Entry B  │
  └──────────┘            └──────────┘
```
