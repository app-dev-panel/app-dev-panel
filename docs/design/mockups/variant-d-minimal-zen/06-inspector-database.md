# Variant D: Minimal Zen — Inspector: Database

## Concept

The database inspector presents schema information as a card browser. Each table is a card.
Cards are stacked vertically and show the table name, row count, column count, and a preview
of the first few columns. Expand a card to see full schema, indexes, and foreign keys.

## Column Type Badges

```
Type annotations shown inline on collapsed cards:

  PK          Primary key           Blue badge
  FK→table    Foreign key           Purple badge with arrow
  unique      Unique constraint     Green badge
  nullable    Nullable              Gray badge, italic
  enum        Enum type             Orange badge
```
