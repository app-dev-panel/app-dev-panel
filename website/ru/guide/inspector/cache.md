# Инспектор кеша

Просмотр, удаление и очистка записей PSR-16 кеша.

## Что показывает

| Возможность | Описание |
|-------------|----------|
| Просмотр | Получение кешированного значения по ключу |
| Удаление | Удаление конкретного ключа кеша |
| Очистка | Сброс всего кеша |

## Использование

Введите ключ кеша для просмотра его сохранённого значения. Инспектор отображает десериализованное значение с информацией о типе.

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/cache?key=my_cache_key` | Просмотр кешированного значения |
| DELETE | `/inspect/api/cache?key=my_cache_key` | Удаление ключа кеша |
| POST | `/inspect/api/cache/clear` | Очистка всего кеша |

## Требования

Требуется реализация PSR-16 <class>Psr\SimpleCache\CacheInterface</class>, зарегистрированная в DI-контейнере.

::: warning
Действие **Очистка** является деструктивным — оно удаляет все записи кеша. Используйте с осторожностью в production.
:::
