# Инспектор переводов

Просмотр и редактирование каталогов переводов для всех локалей.

## Функции

| Функция | Описание |
|---------|----------|
| Браузер каталогов | Просмотр всех каталогов переводов и их сообщений |
| Сравнение локалей | Сравнение переводов между локалями |
| Редактирование | Обновление сообщений переводов прямо из панели |

## Как это работает

Инспектор читает все зарегистрированные источники категорий переводов и отображает сообщения, организованные по каталогам и локалям. Вы можете просматривать сообщения, искать по ключам и редактировать переводы на месте.

## Эндпоинты API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/translations` | Все каталоги и сообщения для всех локалей |
| PUT | `/inspect/api/translations` | Обновление сообщения перевода |

**Тело запроса обновления:**
```json
{
    "catalog": "messages",
    "locale": "en",
    "key": "welcome.title",
    "value": "Welcome to our app"
}
```

## Поддержка адаптеров

Требуется фреймворк-специфичная интеграция переводов (например, Symfony Translator, Laravel Lang). Возвращает 501, если переводы недоступны.

::: tip
Изменения, внесённые через инспектор переводов, обновляют источник переводов приложения напрямую. Это полезно для быстрых исправлений во время разработки.
:::
