# Инспектор авторизации

Просмотр конфигурации безопасности и авторизации вашего приложения.

![Инспектор авторизации](/images/inspector/authorization.png)

## Что показывает

| Раздел | Описание |
|--------|----------|
| Guards | Конфигурация охранников безопасности/файрволов |
| Role hierarchy | Дерево наследования ролей |
| Voters | Зарегистрированные воутеры/политики авторизации |
| Security config | Полный дамп конфигурации безопасности |

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/authorization` | Охранники, иерархия ролей, воутеры, конфигурация безопасности |

## Поддержка адаптеров

| Адаптер | Провайдер |
|---------|-----------|
| Symfony | <class>AppDevPanel\Adapter\Symfony\Inspector\SymfonyConfigProvider</class> (читает конфигурацию `security.yaml`) |
| Прочие | <class>AppDevPanel\Api\Inspector\Authorization\NullAuthorizationConfigProvider</class> (возвращает пустой результат) |

::: info
Инспекция авторизации требует интеграции, специфичной для фреймворка. В настоящее время только адаптер Symfony предоставляет полную интроспекцию конфигурации безопасности.
:::
