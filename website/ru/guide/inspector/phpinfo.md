# PHP Info

Просмотр полного вывода `phpinfo()`, отображённого в панели.

![PHP Info](/images/inspector/phpinfo.png)

## Что показывает

Полная информация о конфигурации PHP, включая:
- Версия PHP и информация о сборке
- Загруженные расширения и их настройки
- Серверный API (SAPI)
- Директивы конфигурации (значения `php.ini`)
- Переменные окружения
- Заголовки HTTP-запросов/ответов

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/phpinfo` | Вывод PHP info |

::: info
Вывод аналогичен прямому вызову `phpinfo()`, отображается как HTML внутри iframe панели.
:::
