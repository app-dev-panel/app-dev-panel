# PHP Info

Просмотр полного вывода `phpinfo()`, отрендеренного в панели.

![PHP Info](/images/inspector/phpinfo.png)

## Что показывает

Полный вывод конфигурации PHP, включая:
- Версию PHP и информацию о сборке
- Загруженные расширения и их настройки
- Server API (SAPI)
- Директивы конфигурации (значения `php.ini`)
- Переменные окружения
- Заголовки HTTP запросов/ответов

## Эндпоинты API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/phpinfo` | Вывод PHP info |

::: info
Вывод идентичен прямому вызову `phpinfo()`, отрендеренный как HTML внутри iframe панели.
:::
