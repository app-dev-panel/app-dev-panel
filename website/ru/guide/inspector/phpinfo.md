# PHP Info

Просмотр полного вывода `phpinfo()`, отображённого в панели.

![PHP Info](/images/inspector/phpinfo.png)

## Что отображается

Полный вывод конфигурации PHP, включая:
- Версия PHP и информация о сборке
- Загруженные расширения и их настройки
- Server API (SAPI)
- Директивы конфигурации (значения `php.ini`)
- Переменные окружения
- Заголовки HTTP-запроса/ответа

## Эндпоинты API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/phpinfo` | Вывод PHP info |

::: info
Вывод аналогичен прямому вызову `phpinfo()`, отображённому как HTML внутри iframe панели.
:::
