# Инспектор OPcache

Просмотр статуса и конфигурации PHP OPcache.

![OPcache Inspector](/images/inspector/opcache.png)

## Что показывает

| Секция | Описание |
|--------|----------|
| Статус | OPcache включён/выключен, использование памяти, процент попаданий |
| Конфигурация | INI-директивы OPcache и их значения |
| Статистика | Попадания, промахи, перезапуски, количество закешированных скриптов |

## Эндпоинты API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/opcache` | Статус и конфигурация OPcache |

::: info
Возвращает HTTP 422, если OPcache не включён на сервере.
:::
