# Инспектор OPcache

Просмотр состояния и конфигурации PHP OPcache.

![Инспектор OPcache](/images/inspector/opcache.png)

## Что отображается

| Раздел | Описание |
|--------|----------|
| Статус | OPcache включён/выключен, использование памяти, процент попаданий |
| Конфигурация | INI-директивы OPcache и их значения |
| Статистика | Попадания в кеш, промахи, перезапуски, количество закешированных скриптов |

## Эндпоинты API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/opcache` | Статус и конфигурация OPcache |

::: info
Возвращает HTTP 422, если OPcache не включён на сервере.
:::
