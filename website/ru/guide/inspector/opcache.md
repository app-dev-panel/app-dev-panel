# Инспектор OPcache

Просмотр состояния и конфигурации PHP OPcache.

![Инспектор OPcache](/images/inspector/opcache.png)

## Что показывает

| Раздел | Описание |
|--------|----------|
| Статус | OPcache включён/отключён, использование памяти, процент попаданий |
| Конфигурация | INI-директивы OPcache и их значения |
| Статистика | Попадания в кеш, промахи, перезапуски, количество кешированных скриптов |

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/opcache` | Состояние и конфигурация OPcache |

::: info
Возвращает HTTP 422, если OPcache не включён на сервере.
:::
