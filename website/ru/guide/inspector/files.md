# Файловый менеджер

Просмотр файловой системы проекта, исходного кода и навигация к определениям классов.

![Файловый менеджер](/images/inspector/files.png)

## Что показывает

| Возможность | Описание |
|-------------|----------|
| Список директорий | Навигация по директориям с метаданными файлов |
| Просмотр файлов | Чтение исходных файлов с подсветкой синтаксиса |
| Разрешение классов | Переход к любому PHP-классу по FQCN |
| Навигация по методам | Переход к начальной/конечной строке конкретного метода |

## Метаданные файла

Каждая запись файла показывает:
- **Имя** и **расширение**
- **Размер** (в байтах)
- **Права доступа** (восьмеричные)
- **Владелец/группа** (имя пользователя или UID)
- **Время последнего изменения**

## Разрешение классов и методов

Укажите полное имя класса (например, `App\Controller\UserController`) и, при необходимости, имя метода для перехода непосредственно к исходному файлу на нужной строке.

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/inspect/api/files?path=/src` | Содержимое директории или чтение файла |
| GET | `/inspect/api/files?class=App\Controller\UserController` | Разрешение класса в файл |
| GET | `/inspect/api/files?class=App\Controller\UserController&method=index` | Разрешение метода класса с диапазоном строк |

## Интеграция с IDE

Пути к файлам маппятся через <class>AppDevPanel\Api\PathMapperInterface</class> для интеграции с IDE. Нажмите на путь к файлу, чтобы открыть его в локальной IDE (VS Code, PhpStorm и т.д.), если настроено.

::: info
Доступ к файлам ограничен корневой директорией проекта. Попытка выйти за пределы корня возвращает ошибку 403.
:::
