### What is ADP?

ADP is the **only** debug panel that works across the entire PHP ecosystem. Install one tool — debug Laravel, Symfony, Yii 3, Yii 2, or any PSR-compatible framework from a single, modern UI.

Unlike Telescope, Symfony Profiler, or Clockwork — ADP is not locked to one framework. **Switch projects, not tools.**

### Why developers choose ADP

- **28 auto-collectors** — logs, SQL, HTTP, events, cache, mail, queues, middleware, exceptions, and more. Zero configuration.
- **Live Inspector** — 20+ pages to X-ray your running app: routes, DB schema, Git, Composer, DI container, OPcache, translations, file explorer. **No competitor has this.**
- **Developer Toolkit** — code generation, cURL builder, request replay, command palette (Ctrl+K), dark mode, PWA.
- **Multi-app & polyglot** — debug microservices from one panel. Send data from PHP, Python, Node.js via OpenAPI 3.1.
- **Real-time** — UDP + SSE streaming. No polling.

### Quick start

```bash
# Laravel
composer require app-dev-panel/adapter-laravel --dev

# Symfony
composer require app-dev-panel/adapter-symfony --dev

# Yii 3
composer require app-dev-panel/adapter-yiisoft --dev

# Yii 2
composer require app-dev-panel/adapter-yii2 --dev
```

Open your app — the debug toolbar appears automatically.
